All the dependencies can be installed with Chocolatey, the package manager for Windows.

  1. choco install -y python2 gtk-runtime microsoft-build-tools libjpeg-turbo (may require sudo/Administrator and reboot).
  2. You will need the cairo library which is bundled in GTK. Download the GTK 2 bundle for Win32or Win64. Unzip the contents in C:\GTK.
  3. Run npm install canvas or yarn add node-canvas with command line from your project folder.
