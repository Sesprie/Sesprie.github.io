Sql Server 2008 R2 初接触

2018-11-16
数据库

下载安装

百度搜索sql server 2008 r2
或直接找到这个页面，如果没有失效的话https://www.microsoft.com/en-us/download/details.aspx?id=3743
也可以直接在这里下载

打开安装的时候，可以更改路径，其他的也可以更改，建议默认，中途会提示登陆模式，可以更改一下sa的默认密码，sa为数据库的默认账号

启动

1.启动服务

方式一：右键我的电脑-->管理-->服务和应用程序-->SQL Server 配置管理器-->SQL Server 服务-->SQL Server (SQLEXPRESS) 启动服务

方式二：命令行输入services.msc，找到sqlserver的服务名称，

​ 这里的服务名称为mssql$sqlexpress，​下次直接net start mssql$sqlexpress或者sc start mssql$sqlexpress

2.启动交互界面

打开SQL Server Management Studio，服务器名称填入(local)\实例名，实例名默认为sqlexpress，如果安装时更改了，对应改一下，身份验证默认为Windows身份验证，也可以选择SQL Server 身份验证，如果安装时更改的sa的密码，直接登陆，否则选择Windows身份验证，然后到(local)\实例名--->安全性--->登陆名--->右键sa--->属性--->常规更改密码，重复两次新密码即可

命令行登陆

两个命令osql和sqlcmd，区别在sqlcmd支持的功能多

1.登陆

sqlcmd -S (local)\sqlexpress -U sa -P 密码

2.新建数据库并插入数据

3.删除数据库
